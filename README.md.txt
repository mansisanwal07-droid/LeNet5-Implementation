# LeNet-5 Research Paper Implementation
Implemented by **Mansi Sanwal**

This project is a from-scratch implementation of the classic **LeNet-5** architecture as described in the 1998 paper *"Gradient-Based Learning Applied to Document Recognition"*.

## Project Overview
- **Architecture:** 2 Convolutional layers, 2 Pooling layers, 3 Fully Connected layers.
- **Dataset:** MNIST Handwritten Digits.
- **Framework:** PyTorch.

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Run training: `python train.py`
